Overview
========
Currently reads files from "SMARTER" folder on the sdcard
Contents of this folder is generated from the Export fuction from the SMARTER eVCS webapp (https://github.com/thewheat/SMARTER-eVCS-webapp/)


Development
===========
Requires:
-Android SDK http://developer.android.com/sdk/
-ADT Plugin (if using Eclipse) http://developer.android.com/tools/sdk/eclipse-adt.html

Import project into Eclipse
- File > Import > Existing Projects into Workspace
- Specify directory in "Select root directory"
- Click Finish and run project as an Android application
